## 📖 Описание проекта

**Random Password Generator** — это графическое приложение для генерации случайных паролей с возможностью настройки параметров, сохранением истории и анализом сложности создаваемых паролей.

### Зачем это нужно?
- Создание надёжных паролей для сайтов и сервисов
- Быстрая генерация паролей заданной сложности
- Хранение истории для последующего использования
- Анализ статистики используемых паролей

---

## 🎯 Функциональные возможности

| Функция | Описание |
|---------|----------|
| **Генерация пароля** | Создание случайного пароля с заданными параметрами |
| **Настройка длины** | Ползунок от 4 до 32 символов |
| **Выбор символов** | Строчные/заглавные буквы, цифры, спецсимволы |
| **Оценка сложности** | Автоматический расчёт надёжности пароля |
| **История паролей** | Таблица со всеми сгенерированными паролями |
| **Копирование** | Один клик для копирования пароля в буфер обмена |
| **Статистика** | Анализ использования: средняя длина, распределение по сложности |
| **Сохранение** | Автоматическое сохранение истории в JSON |
